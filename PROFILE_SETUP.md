# GitHub Profile README Setup

## How to use

1. Create a new GitHub repository named exactly the same as your GitHub username.
2. Copy `README.md` from this folder into that repository root.
3. The GitHub username in this version is already set to `lzx54`.
4. Push the repository, and GitHub will show it on your profile page.

## Optional upgrades

- Add a banner image at the top.
- Add links to your best 3 to 6 projects.
- Add a portfolio link if you have one.
- Add a `waka-readme` or contribution snake only if you want motion, not just decoration.
